## Criando Meu primeiro Bot

Adicione a conta ```botFather``` no seu Telegram. Este bot exibirá os comandos para voce criar o bot. 

    @botfather

Crie o bot com o nome desejado

BotName   ```ParabensAmigo```
UserName  ```ParabensToYoubot```
